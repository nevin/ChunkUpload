# PHP server for Chunk Upload

## Capablities 
1. 	Recieves chunks uploaded from the UI
1. 	Validates & combines the chunk in the server
1. 	Provides the list and properties of the files uploaded
	
	
## Libraries Used 
*  getID3  - A PHP media Parse library
